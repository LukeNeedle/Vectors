# Vectors
 Basic vector drawing

## Files
| File          | Description      | File contents                                                             |
|---------------|------------------|---------------------------------------------------------------------------|
| import.CONF   | Imports settings | Line 1 = Drawing speed <br> Line 2 = Image Scale <br> Line 3 = Line width |
| import.VECTOR | Imports vectors  | 3D list of vectors [x,y]                                                  |
